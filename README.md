# netease music message history

This project uses Selenium to scrape private message history from NetEase Cloud Music.

## Requirements

- Python 3
- Google Chrome
- Selenium
- lxml

## Files

- [login.py](./login.py): opens NetEase Cloud Music and records login data in a local Chrome profile.
- [message.py](./message.py): opens the private message page and writes message history to a text file.
- [.gitignore](./.gitignore): ignores local personal data generated by this project.

## Usage

Run [login.py](./login.py) first:

```powershell
python login.py
```

After the browser opens, log in to NetEase Cloud Music manually. The login profile will be saved here:

```text
selenium_chrome_profile/
```

Then run [message.py](./message.py):

```powershell
python message.py
```

If the script asks you to select a friend, select the friend in the browser window and wait for the script to continue.

## Output

The scraped message history is written to:

```text
msghistory/history.txt
```

## Edit Display Names

If you want to replace a sender name in the output, edit the name replacement code in [message.py line 35](./message.py#L35).

Current placeholder code:

```python
if sender == "origin name":
    sender = "new name"
```

Change `"origin name"` to the original sender name, and change `"new name"` to the name you want to write into `history.txt`.

## Notes

- `selenium_chrome_profile/` and `msghistory/` are local personal data folders. Do not commit login data or scraped private message history to GitHub.
